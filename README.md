# Retail-Trial-vs-Control-Store-Analysis

# Overview

This project focuses on identifying statistically comparable control stores for selected trial stores and measuring the impact of retail interventions using sales uplift analysis.

 # Methodology

Performed data preprocessing and time-series alignment at monthly level

Used correlation analysis and similarity metrics to identify optimal control stores

Calculated expected sales using scaling factors and compared against actual trial performance

Measured uplift % to evaluate effectiveness of trial initiatives

# Key Insights

All trial stores showed positive uplift compared to control stores

Store 86 achieved the highest uplift (~12.9%), indicating strong intervention impact

Product-level insights revealed high demand for specific brands and pack sizes

# Tools & Technologies

Python (Pandas, NumPy), Excel, Data Visualization
